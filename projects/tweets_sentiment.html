Data Visualization using D3



Languages :

Python, JavaScript

duration :

December 2022 - Now

links :

None for the moment

This project involves building a real-time sentiment analysis pipeline for tweets using natural language processing techniques. The pipeline includes collecting tweets using the Twitter API, pre-processing the text data, and applying sentiment analysis models to classify tweets as positive, negative or neutral. The pipeline will be implemented using programming languages such as Python, and libraries like NLTK, scikit-learn, and pandas. The real-time aspect of this project is achieved by using a message queue such as Apache Kafka or RabbitMQ to stream tweets and process them in real-time. The analyzed data will then be visualized using D3.js, a JavaScript library for creating interactive data visualizations. The project is currently in progress and not finished yet, with a focus on real-time data processing and visualization.

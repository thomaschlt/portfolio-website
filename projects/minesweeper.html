"Silver Eureka" is a game inspired by the classic Minesweeper game, but with a unique twist. In this game, players will navigate a grid of squares, trying to uncover hidden treasure while avoiding dangerous traps. The game is developed using C# and the game grid is generated randomly with treasure and traps placed in different positions each time the game starts. The player will have to use logic and strategy to uncover squares and find the treasure while avoiding the traps. The game features different levels with increasing difficulty as the player progresses through the game. The player can also earn points for each successful treasure found and avoid penalty for triggering a trap. The player can also save their progress and come back to the game later. As my first project in C#, I learned implementing game logic.

It was a great learning experience and I had a lot of fun developing it !
